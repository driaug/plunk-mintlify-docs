You can use the Bubble plugin to send events and emails from your Bubble apps.

## Getting Started
Install the plugin from the [Bubble plugin store](https://bubble.io/plugin/plunk-%7C-email-platform-for-saas-1656752337631x320149042131370000).

Once installed, the plugin provides two actions:
- `Send event to Plunk`
- `Send email with Plunk`
